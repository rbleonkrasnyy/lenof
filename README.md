# LenOf

## How to use

### Node.js

```javascript
const lenof = require('./lenof.js');

const len = lenof({
    name: "Leonid Krasovsky",
    message: "I love Natalie",
    reason: "She's pretty",
});

console.log(len);
```

Result:

```
3
```
